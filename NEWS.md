# NEWS

## v0.1.0

- Første offentlige release av Shiny-appen for pedagogisk overgang mellom ICD-10-typer og ICD-11-lignende trekkprofiler.
- Appen bruker en delt modell med tunede vekter, toleranser og bias-filer ved kjøring.
- Kalibreringsskriptet gjør det mulig å teste og tune modellen mot faglige ankerprofiler.
- GitHub Actions-workflow for deploy er på plass.
- Modellen er pedagogisk og skal ikke brukes diagnostisk.
